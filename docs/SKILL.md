## Quick Reference

### Colors
- **Rebellions Green**: #52F756 (the core)
- **Gray 90 (Dark)**: #1B1F23
- **Gray 80**: #24292F
- **Gray 10**: #F6F8FA
- **White**: #FFFFFF
- **Black**: #000000
- Secondary: Blue #174BEB, Yellow #FFD527, Purple #9A4EFF, Pink #F7318B, Red #FF3333

### Typography
- **Söhne** — English headlines (weights 200–900)
- **Pretendard** — Korean body/headlines (weights 200–900)
- **Space Mono** — Tech labels, code, captions (400, 700)

### Visual Rules
- Border radius: always 0px (sharp corners)
- Borders: 0.5px solid rgba(0,0,0,0.08)
- Dark backgrounds dominant
- No emoji, no rounded corners
- Hover: opacity 0.85 or translateY(-2px)
