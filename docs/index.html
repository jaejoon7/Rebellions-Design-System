Usage Guidelines
Clear Space

Maintain a minimum clearance equal to the symbol height around the logo. No graphic elements or text should encroach on this area.

Minimum Size

Digital: minimum width 120px. Print: minimum width 30mm. Below these sizes, legibility cannot be guaranteed.

Background

Use the Negative (white) logo on dark backgrounds and the Positive (black) logo on light backgrounds.

Don'ts

Do not alter logo colors, distort proportions, add shadows or effects, rotate, or rearrange logo elements.

03 — COLOR & GRADIENT

Color & Gradient

Our color palette expands on the Rebellions aesthetic—neon green at the core, balanced with precise neutrals and purposeful accent colors.

Green at the core

Rebellions Green is the centerpoint of the color palette. Combined with the precision of black and white,
it creates the signature identity that defines every Rebellions touchpoint.

The palette

The full palette extends from the neutral gray family to the secondary color spectrum. Each experience should be dominated by the grays and the core colors—black, white, and Rebellions Green—allowing the secondary colors to provide vibrancy and purpose.

Grays

The neutral gray scale provides nuance and meaningful moments of color.
Five grays from F6F8FA to 1B1F23 cover the full range of backgrounds and text, ensuring every design maintains legibility and hierarchy.

04 — BRAND EXPRESSION

Brand Expression

An isometric cube grid system that forms the visual identity of Rebellions.

Adjust grid size, motion, neon effects, angles, and more to generate brand graphics exportable as SVG/PNG/GIF.
AX Prompt Input
Describe the expression in plain language. Product words — chip · card · server · rack · data center · system — set the base scale-up. Tone words (비즈니스·에너지·기본) tune neon, motion words (웨이브·호흡·펄스·정적) set the animation, and scale words (웅장·타워·칩) adjust size — plus modifiers (denser / taller / wireframe) and explicit counts (12×12, gap 20). When no keyword is recognized, the grid is left unchanged.
Prompt

Type a prompt and press Generate to drive the grid above.
Controls Guide
SCALE-UP

Adjust cube height. Scale from chip to tower.

GRID

Set grid size, spacing, and view angle. Adjustable from 10° (side) to 60° (top).

MOTION

Four motion presets: Breathe, Wave, Pulse, and Static.

STYLE

Fine-tune deformation, neon ratio, speed, stroke width, opacity, and more.

EFFECTS
